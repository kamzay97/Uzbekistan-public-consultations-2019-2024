# Uzbekistan-public-consultations-2019-2024
Dataset and reproducible R analysis scripts for the CPRO Analytical Report Revisiting the Online Public Consultation Model in Uzbekistan (2026). Covers 4,648 draft regulations posted on regulation.gov.uz between 2019 and 2024.

This repository contains the data and reproducible code underlying the CPRO Analytical Report Revisiting the Online Public Consultation Model in Uzbekistan: A Six-Year Evidence Review, 2019–2024, published by the Center for Policy Research and Outreach (CPRO) at Westminster International University in Tashkent. The dataset covers 4,648 draft regulations posted for public comment on regulation.gov.uz — Uzbekistan's official online regulatory consultation portal — between January 2019 and December 2024. The R scripts reproduce all tables and statistical indicators presented in the report, including comment distribution metrics, developer participation dynamics, and regulatory type over-representation ratios.
